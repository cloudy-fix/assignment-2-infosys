# Infosys RAG Compliance Checker V1

AI-assisted regulatory compliance checker assignment focused on contract review, risk identification, and document intelligence workflows.

## Interview Angle

This repository is useful for discussing how LLM and RAG-style systems can support contract analysis without replacing human review. It shows practice in turning unstructured legal text into a more reviewable workflow.

## What To Discuss

- Contract/document ingestion and preprocessing
- Clause and risk identification flow
- Compliance-oriented prompt and retrieval design
- Python implementation decisions
- How results can be explained to a reviewer

## Skills Demonstrated

- Python
- LLM workflow design
- RAG concepts
- Regulatory compliance analysis
- Document automation
